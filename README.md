# PAYROLL-COBOL-CLAUDE

> ## 🧪 Research artifact of the coding-agents COBOL study
>
> This repository is **`PAYROLL-COBOL-CLAUDE`** (folder `cobol-jb-cc`), one of **16 COBOL systems
> developed from scratch by frontier coding agents** in the multi-case study
> *Coding Agents Develop COBOL Systems: Evidence from a Multi-Case Empirical Study* (2026).
>
> **What it is:** a payroll batch system built under an externally authored six-step protocol.
> **Oracle:** per-step structural criteria and a behavior-preserving refactoring verified by differential execution (byte-identical outputs).
> **Built by:** Claude Code (`claude-sonnet-4-6`) in recorded sessions: 21 substantive prompts,
> 2.0 active hours, 535 lines of COBOL, difficulty High.
>
> The code, documentation, and git history are preserved as the agent produced them
> (treat them as research artifacts). Study hub with all 16 systems, the analysis
> pipeline, session logs, and replay packs: **https://github.com/acherm/agentic-cobol-study**


## Building

Requires GnuCOBOL (`cobc`). See the Makefile or scripts in this repository, and the replay pack in the study hub for the exact prompts and validation criteria per activity.
